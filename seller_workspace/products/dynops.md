# Product: DynOps (Dynamic Operations Analysis)

## Description
Real-time and post-processing analysis platform for dynamic positioning (DP) operations, covering DP capability assessment, SIMOPS planning, and operational envelope definition.

## Target Use Cases
- DP vessel / MODU capability assessment prior to campaign
- SIMOPS planning for concurrent DP operations
- DP incident analysis and root cause investigation
- Regulatory compliance (IMO, DNVGL, MCA)

## Ideal Customer Profile
- Operator running DP-assisted drilling or intervention campaigns
- Drillship or semi-submersible rig with DP2/DP3 classification
- FPSO with DP-offloading requirement
- Operator planning multi-vessel SIMOPS in congested field

## Technical Fit Signals (news triggers)
- "drillship", "semi-submersible", "DP2", "DP3", "dynamic positioning"
- "SIMOPS", "concurrent operations", "multi-vessel"
- "FPSO", "offloading", "mooring"
- Environmental conditions: harsh, variable current, cyclone season

## Timing Signals
- Rig contract awarded (3–6 month window before mobilization)
- FPSO first-oil announcement (production optimization phase)
- Campaign planning / FEED stage
- Post-incident investigation

## Disqualifiers
- Fixed platform without DP vessel involvement
- Jack-up (moored, not DP)
- Onshore operation
