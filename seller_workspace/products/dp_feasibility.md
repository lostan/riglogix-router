# Product: DP Feasibility Study

## Description
Pre-campaign feasibility study assessing whether a DP-assisted operation can be safely conducted at a given location, given environmental conditions, vessel capability, and regulatory requirements.

## Target Use Cases
- Pre-campaign DP feasibility for new field / new vessel combination
- Environmental exceedance analysis (weather downtime estimation)
- Regulatory DP FMEA support
- New geography entry requiring DP envelope definition

## Ideal Customer Profile
- Operator entering a new field or basin with DP operations
- Operator contracting a new DP drillship or semi
- Operator facing regulatory request for DP feasibility documentation
- Project team in FEED/pre-FEED with open DP operability question

## Technical Fit Signals (news triggers)
- "new field development", "first well", "new block", "license award"
- "DP vessel", "drillship", "semi-submersible"
- "environmental study", "metocean", "operability"
- Regulatory body mentions: ANP, BSEE, NORSOK, MCA
- New geography: frontier basin, new country entry

## Timing Signals
- License / block award (12-18 month planning window)
- Vessel selection / contracting phase
- FEED initiation for new development
- Regulatory submission deadline

## Disqualifiers
- Operation already underway (post-campaign)
- Fixed platform (no DP vessel)
- Repeat operation in known field with existing feasibility study (unless conditions changed)
